# Piped Frontend

Альтернативный фронтенд для YouTube с использованием Piped API. Позволяет искать и смотреть видео без рекламы, трекинга и необходимости аккаунта Google.

![Piped Frontend](https://img.shields.io/badge/version-1.0.0-red.svg)
![License](https://img.shields.io/badge/license-MIT-blue.svg)

## 🎬 Особенности

- **Поиск видео и каналов** - Поиск по всем категориям (видео, каналы, плейлисты)
- **Тренды** - Просмотр популярных видео
- **Просмотр видео** - Полноценный плеер с информацией о видео
- **Комментарии** - Просмотр комментариев к видео
- **Страницы каналов** - Информация о канале и его видео
- **Автоматическое переключение плееров** - 3 скрипта плеера с автоматическим переключением при ошибке
- **Несколько Piped инстансов** - Автоматический выбор рабочего инстанса
- **Тёмная тема** - Приятный дизайн в стиле YouTube Dark
- **Адаптивность** - Работает на всех устройствах

## 🔧 Конфигурация

### Piped Инстансы

В файле `app.js` можно настроить список используемых Piped инстансов:

```javascript
const CONFIG = {
    instances: [
        'https://pipedapi.kavin.rocks',
        'https://pipedapi.in.projectsegfau.lt',
        'https://pipedapi.adminforge.de',
        'https://api.piped.yt'
    ],
    // ...
};
```

Приложение автоматически выбирает первый рабочий инстанс.

### Скрипты Плеера

Настройте приоритет скриптов плеера:

```javascript
const CONFIG = {
    playerScripts: [
        'https://cdn.jsdelivr.net/gh/OinkTechLLC/cdnplayerjs@main/playerjs.js',
        'https://cdn.jsdelivr.net/gh/OinkTechLtd/cdnplayerjs@main/playerjs.js',
        'https://cdn.jsdelivr.net/gh/twixoffltdco/cdnplayerjs@main/playerjs.js'
    ]
};
```

Если первый скрипт не работает, приложение автоматически переключится на следующий.

## 🎮 Использование

### Поиск видео

1. Введите запрос в поисковую строку
2. Выберите тип поиска (Всё, Видео, Каналы, Плейлисты)
3. Нажмите "Поиск" или Enter

### Просмотр видео

1. Кликните на карточку видео в результатах поиска или трендах
2. Видео откроется в плеере
3. Используйте кнопки для переключения между плеерами если текущий не работает

### Просмотр канала

1. Кликните на карточку канала
2. Откроется страница канала с информацией и видео

### Переключение плеера

В правом верхнем углу плеера есть кнопки для переключения между доступными плеерами:
- **Плеер 1** - OinkTechLLC
- **Плеер 2** - OinkTechLtd  
- **Плеер 3** - twixoffltdco

Если плеер не загружается, система автоматически переключится на следующий.

## 🌐 API

Проект использует Piped API для получения данных:

- `/trending` - Получение трендов
- `/search?q={query}` - Поиск
- `/streams/{videoId}` - Информация о видео
- `/comments/{videoId}` - Комментарии
- `/channel/{channelId}` - Информация о канале

Документация Piped API: https://docs.piped.video/docs/api-documentation/
