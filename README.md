# 📺 TikTokClone - YouTube Viewer с поддержкой VPN

Приложение для просмотра YouTube видео в стиле TikTok с поддержкой просмотра из России, КНР и других стран через API ключ.

## 🔥 Возможности

- ✅ Просмотр трендовых видео YouTube
- ✅ Поиск видео и каналов
- ✅ Работает из любой страны (Россия, Китай и др.)
- ✅ Интерфейс в стиле TikTok
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Сохранение API ключа в браузере

---

## 📋 Требования

1. Аккаунт Google (любой Gmail)
2. Браузер (Chrome, Firefox, Safari, Edge)
3. 5-10 минут свободного времени

Для безопасности рекомендуется ограничить использование ключа:

1. Нажмите **"РЕДАКТИРОВАТЬ КЛЮЧ"** (EDIT KEY)
2. В разделе **"Ограничения приложений"** выберите **"HTTP-рефереры"**
3. Добавьте: `localhost`, `127.0.0.1`, `*.github.io`
4. В разделе **"Ограничения API"** выберите **"Ограничить ключ"**
5. Отметьте только **"YouTube Data API v3"**
6. Нажмите **"СОХРАНИТЬ"**

### Вариант 2: Прямое редактирование файла

1. Откройте файл `index.html` в текстовом редакторе
2. Найдите строку (примерно строка 424):
   ```javascript
   this.apiKey = localStorage.getItem('youtube_api_key') || '';
   ```
3. Замените на:
   ```javascript
   this.apiKey = 'ВАШ_API_КЛЮЧ_ЗДЕСЬ';
   ```
4. Сохраните файл и откройте в браузере

## ❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ

### Q: Это бесплатно?
**A:** Да! YouTube Data API предоставляет бесплатную квоту:
- 10,000 единиц в день бесплатно
- Просмотр одного видео = 1 единица
- Хватит на ~10,000 просмотров в день

### Q: Что если квота закончится?
**A:** Просто создайте новый проект в Google Cloud и получите новый ключ.

### Q: Безопасно ли это?
**A:** Да, ключ хранится только в вашем браузере (localStorage) и никуда не отправляется.

### Q: Можно ли использовать на телефоне?
**A:** Да! Приложение полностью адаптировано для мобильных устройств.

### Q: Почему видео не загружаются?
**A:** Проверьте:
1. Правильность API ключа
2. Активирован ли YouTube Data API v3
3. Есть ли интернет-соединение
4. Не истекла ли квота

## 🔧 УСТРАНЕНИЕ НЕПОЛАДОК

### Ошибка "API Error: 400"
- Проверьте правильность ключа API
- Убедитесь, что YouTube Data API v3 активирован

### Ошибка "API Error: 403"
- Ключ не имеет доступа к YouTube API
- Проверьте ограничения ключа

### Ошибка "API Error: 429"
- Превышена дневная квота
- Подождите до завтра или создайте новый ключ

### Видео не загружаются
1. Откройте консоль браузера (F12)
2. Посмотрите ошибки
3. Проверьте API ключ в настройках
